Bootstrap Theming Template

About this template

This template has been created for the 2015 Moodle Moot theming workshop. It can be used to customize Moodle HTML and CSS without having to install Moodle on your machine. This template is based on the Bootstrap parent theme.

How to customize this template

To customize this template you will need a code editor. I recommend using the Atom editor if you do not have a preferred editor.

Template content

This template contains:

  • This home page
  • A course page
  • The myhome page
  • A Moodle quiz

Further instruction on customizing this template will be provided during the workshop.
