About this folder
=================

Use the custom.css file to add your custom styling for Moodle.

If you want to use your custom.css simply copy-paste it into course customcss box on the theme settings page in Moodle.

WARNING!
The all.css contains a Modified version of Bootstrap and Moodle css for this specific template. Do not use it on a real Moodle theme.
